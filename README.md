CSparseJ is a Java port of [CXSparse](https://github.com/DrTimothyAldenDavis/SuiteSparse/tree/dev/CXSparse)

### CSparseJ 1.2

[CSparseJ-1.2.jar](https://repo1.maven.org/maven2/com/github/wendykierp/CSparseJ/1.2/CSparseJ-1.2.jar) 

[CSparseJ-1.2-javadoc.jar](https://repo1.maven.org/maven2/com/github/wendykierp/CSparseJ/1.2/CSparseJ-1.2-javadoc.jar) 

[CSparseJ-1.2-sources.jar](https://repo1.maven.org/maven2/com/github/wendykierp/CSparseJ/1.2/CSparseJ-1.2-sources.jar) 

CSparseJ is available on maven central as

    <dependency>
        <groupId>com.github.wendykierp</groupId>
        <artifactId>CSparseJ</artifactId>
        <version>1.2</version>
    </dependency>
